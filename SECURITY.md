# Security Policy

## Supported Versions

This project is pre-1.0. Security fixes target the current `main` branch.

## Reporting a Vulnerability

Please do not open a public issue for sensitive security problems.

Use GitHub's private vulnerability reporting feature for this repository if it
is available. If it is not available, contact the repository owner privately and
include:

- A concise description of the issue.
- Steps to reproduce it.
- The affected platform and desktop environment.
- Whether the issue requires a malicious local file, clipboard content, shell
  command, or terminal program.

This app launches local shells and terminal programs. Reports are most useful
when they identify behavior in `linux-cmd-dashboard` itself rather than normal
behavior of the shell or programs running inside a pane.
